# Linking Real-Time Construction Productivity Analytics to Scheduling and Control

This repository accompanies the manuscript:
**Linking Real-Time Construction Productivity Analytics to Scheduling and Control: A Closed-Loop Framework for Adaptive Production Management**

## Contents
- `data/`: dataset files used by the manuscript
- `scripts/make_figures.py`: regenerates key paper figures
- `scripts/run_summary.py`: prints the main analytical and project-level results
- `figures/`: publication-ready figures
- `docs/suggested_insert_text.md`: suggested manuscript text for GitHub/data-code availability and figure captions

## Main reported project-level results
- Average PPC, manual: **52.2%**
- Average PPC, monitoring only: **56.4%**
- Average PPC, closed-loop: **66.5%**
- Final cumulative slip, manual: **789.0**
- Final cumulative slip, monitoring only: **719.0**
- Final cumulative slip, closed-loop: **554.6**
